## üîß Troubleshooting

### Common Issues

**Workflow not triggering:**
- Ensure workflows are on the default branch
- Check event trigger configuration
- Verify branch name matches trigger

**Azure authentication failing:**
- Verify service principal credentials
- Check secret names match workflow
- Ensure SP has correct permissions

**Container action not working:**
- Test Docker image locally first
- Check Dockerfile syntax
- Verify entrypoint script is executable

**GitHub Actions minutes exhausted:**
- Use ubuntu-latest (cheapest)
- Implement path filtering
- Cache dependencies

### Getting Help

1. Review lab-specific troubleshooting sections
2. Check workflow run logs in Actions tab
3. Search GitHub Actions documentation
4. Ask in GitHub Community Forum
